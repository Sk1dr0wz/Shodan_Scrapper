# Shodan_Scrapper

# Installation
```
git clone https://github.com/Sk1dr0wz/Shodan_Scrapper.git
cd Shodan_Scrapper
pip -r requirements.txt
```

# API
```
You need to fill yours Shodan API in config.ini before run the tool .
```

# How To Run
```
python3 dork.py
```
